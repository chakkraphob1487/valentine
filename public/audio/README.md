# Audio Files Placeholder

เว็บไซต์ต้องการไฟล์เสียง 3 ไฟล์:

1. **background_music.mp3** - เพลงประกอบพื้นหลัง
   - แนะนำ: เพลงโรแมนติก instrumental หรือ love song
   - ความยาว: 2-3 นาที (loop ได้)

2. **button_click.mp3** - เสียงเมื่อกดปุ่ม
   - แนะนำ: เสียง "pop" หรือ "click" สั้นๆ
   - ความยาว: < 1 วินาที

3. **heart_sound.mp3** - เสียงหัวใจเต้น
   - แนะนำ: เสียงหัวใจเต้น หรือเสียง "ding" น่ารักๆ
   - ความยาว: < 2 วินาที

## วิธีเพิ่มไฟล์เสียง:

1. หาไฟล์เสียงที่ต้องการ (จาก freesound.org, YouTube Audio Library, หรือแหล่งอื่น)
2. แปลงเป็นไฟล์ .mp3
3. วางไฟล์ในโฟลเดอร์ `public/audio/`
4. ตั้งชื่อไฟล์ตามที่ระบุด้านบน

**หมายเหตุ:** เว็บไซต์จะทำงานได้ปกติแม้ไม่มีไฟล์เสียง แต่จะไม่มีเสียงประกอบ
